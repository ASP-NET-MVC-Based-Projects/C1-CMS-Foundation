Setting up Grunt with Visual Studio 2015

  1. You need to install Node.JS which you can get from here https://nodejs.org/en/download/
  2. You need to install Grunt globally which you do by running this command

    npm install -g grunt-cli

    You can do that on your command shell as Administrator (for Windows). For more details read here - http://gruntjs.com/getting-started
  3. Make sure that you have Web Essentials installed for Visual Studio
    http://vswebessentials.com/

    You need to restart Visual Studio in order to get these extensions working.

  4. Open "Task Runner Explorer". In Visual Studio open
    VIEW -> Other Windows -> Task Runner Explorer
  5. On "Task Runner Explorer" window press "Refresh" button and run "build" task.
  6. Build CompositeC1 solution.
  7. You are done. CSS is compiled.
    Start Composite C1 console and press F5 in your Browser.
