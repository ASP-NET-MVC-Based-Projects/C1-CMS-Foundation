Setting up Grunt with Visual Studio 2013

  1. You need to install Node.JS which you can get from here http://nodejs.org/download/
  2. You need to install Grunt globally which you do by running this command

    npm install -g grunt-cli

    You can do that on your command shell as Administrator (for Windows). For more details read here - http://gruntjs.com/getting-started
  3. Install Grunt Runner for Visual Studio
    https://visualstudiogallery.msdn.microsoft.com/8e1b4368-4afb-467a-bc13-9650572db708
  4. Install Grunt Launcher for Visual Studio (additional features for Grunt, Gulp, npm and Bower) -
    https://visualstudiogallery.msdn.microsoft.com/dcbc5325-79ef-4b72-960e-0a51ee33a0ff
  5. Opent the C1 solution in Visual Studio. In Website root folder locate the file
    package.json -> Right click -> NPM install packages.
    That command will add next folder "node_modules".
  6. Open "Task Runner Explorer". In Visual Studio open
    VIEW -> Other Windows -> Task Runner Explorer
  7. On "Task Runner Explorer" window press "Refresh" button and run "build" task.
  8. You are done. CSS is compiled.
    Start Composite C1 console and press F5 in your Browser.
